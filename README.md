##  住转 砖 (Snakes and Ladders)

砖拽  拽住 注专 2-4 砖拽.

### 转转:
-  转 -2-4 砖拽
-  转 拽 注 爪
-  住转 砖注 转 注
-  砖 砖专 转 
-  注爪 爪注 砖
-  注拽 专 拽  砖拽

###  砖拽:
1. 驻转 转 拽抓 `snakes-and-ladders.html` 驻驻
2. 专 住驻专 砖拽 (2-4)
3. 抓 注 " 拽" 转专
4. 砖拽 专砖 砖注 砖爪转 100 爪!

**驻:** 住 注 住转 注 砖!

##  砖拽 砖 (Snake Game)

砖拽 住拽 拽住 住 注 拽 驻砖.

### 转转:
-  砖 爪注转 拽砖 爪
-  注拽 专 拽 砖 砖
- 革 驻砖专转 砖 (拽砖 专)
-  注爪 专 砖
-  砖专转 砖 砖拽

###  砖拽:
1. 驻转 转 拽抓 `snake.html` 驻驻
2. 砖转砖 拽砖 爪 转 砖
3. 住祝 转    爪专 拽转
4. 注 转砖转 拽专转  祝 砖

**驻:** 抓 专  砖转 转 砖拽!
